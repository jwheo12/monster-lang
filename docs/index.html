LLVM-first AOT systems language

Monster documentation, kept simple and close to the compiler.

Monster is an experimental low-level language with a Rust-flavored surface, explicit control, LLVM IR output, and a small toolchain built around the mst compiler.

Install Monster Read the language guide

Pipeline

.mnst source
  -> lexer
  -> parser
  -> AST
  -> semantic analysis
  -> LLVM IR
  -> opt-18 (-O2)
  -> clang-18
  -> native binary
  • Compiler executable: mst
  • Source extension: .mnst
  • Static docs ready for GitHub Pages and editor help links

    • Current Feature Snapshot

    Language Core

    fn, extern fn, let, let mut, if, while, break, continue, and return/return;.

    Data Model

    i32, u8, usize, bool, str, void, fixed arrays, slices, structs, pointers, enums, and payload enums.

    Low-Level Utilities

    sizeof(T), argc/argv, file I/O builtins, string and byte helpers, and explicit casts with as.
